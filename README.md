# Backrooms Escape Together - USMAP Research

## Overview

This repository contains a custom `.usmap` mapping file used for Unreal Engine asset structure decoding and research purposes for the game **Backrooms Escape Together**.

The file was extracted and generated for analysis using tools such as FModel and custom Unreal Engine mapping workflows.

---

## Game Information

* **Game:** Backrooms Escape Together
* **Game Version:** `0.14.1`
* **Engine Version:** Unreal Engine `5.7`

---

## Contents

* `*.usmap` → Unreal Engine property and struct mappings for asset decoding

No game assets (models, textures, audio, or levels) are included in this repository.

---

## Purpose

This repository is intended for:

* Unreal Engine asset research
* FModel compatibility and decoding
* Modding / reverse engineering study
* Understanding game structure and asset layout

---

## Usage

### In FModel:

1. Open FModel
2. Load game directory
3. Import the `.usmap` file
4. Enable custom mappings
5. Browse assets with readable structure
